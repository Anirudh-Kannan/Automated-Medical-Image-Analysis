  • MediSync

    MediSync is a state-of-the-art platform that uses deep learning to automatically find and classify a broad spectrum of diseases and abnormal signs in chest radiographs.

  • The most widespread method interpretation of medical data such as radiology reports are done manually by medical experts. This is however quite limited due to its subjectivity, complexity of the image, extensive variations that exist across different interpreters, bias and fatigue.

  • We are in the middle of a revolution of Artificial Intelligence that is rapidly changing the field of medical image analysis. This platform combines the best of AI and X-ray image processing to help medical professionals in the diagnosis of ther patients' chest X-rays.

Our Project.

A ResNet based deep learning model is fine tuned on a dataset consisting of 5863 samples of Chest X-rays is used to extract relevant features required to perform classification
T-SNE is used to determine the clusters using unsupervised machine learning techniques. Finally a softmax layer is used to classify these features. The output of this neural network will indicate whether the patient has a disease or not.
Using the Flask framework in Python, we create a REST API that takes an X-ray image as input and returns the results from our model in JSON format. This API can now be called in our web application to analyze the uploaded images.
